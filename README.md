# 🛰️ Satellite Image Land-Use Classifier & Temporal Change Detector

A computer vision project that classifies satellite land-use imagery and
detects temporal land-cover changes using **transfer learning** and
**embedding similarity**, with an interactive **Streamlit dashboard**.

---

## ✨ Features

- 🌍 Land-use classification using a two-phase fine-tuned **ResNet18**
- 🥇 Top-3 predictions with confidence scores, for both tiles
- 🔄 Temporal change detection using **512-dimensional embeddings**
- 📊 ROC-based cosine similarity threshold selection
- 🎛️ Multi-threshold detection mode — switch between **High Recall**,
  **Balanced**, and **High Precision** operating points
- 🔥 Pixel-difference change heatmaps
- 📈 Baseline scratch CNN vs. transfer learning comparison
- 🧪 Spatial leakage experiment (block split vs. random split)
- 📉 Error analysis on the top-5 misclassified class pairs

## 📚 Datasets

### EuroSAT
- 27,000 RGB satellite images, 10 land-use classes
- Downloaded automatically inside the notebooks:
  ```python
  torchvision.datasets.EuroSAT(download=True)
  ```

### UC Merced Land Use
- 2,100 RGB images, 21 land-use classes
- Downloaded automatically inside Notebooks 3 and 5

## 🚀 Running the Notebooks

All notebooks are designed for **Google Colab** — open each one and use
**Runtime → Run all**. Run them in order:

1. **`01_Data_Pipeline.ipynb`** — dataset download, class distribution,
   sample tiles, spatial block split.
2. **`02_Baseline_CNN.ipynb`** — scratch 3-layer CNN, loss curves,
   per-class F1.
3. **`03_Transfer_Learning.ipynb`** — two-phase ResNet18 fine-tuning
   (frozen head → unfrozen last 2 blocks), frozen-vs-unfrozen comparison,
   EuroSAT confusion matrix, UC Merced evaluation. Saves the checkpoint as
   `models/model.pt`.
4. **`04_Temporal_Change_Detection.ipynb`** — embedding extraction, cosine
   similarity, ROC curve, threshold selection, 5 change heatmaps.
5. **`05_Evaluation.ipynb`** — spatial leakage experiment (block vs. random
   split), full UC Merced per-class metrics table, top-5 misclassified
   pairs with example images.

## 💻 Running the Streamlit Dashboard

The dashboard runs locally (not in Colab):

```bash
pip install -r requirements.txt
```

Place your trained checkpoint at:

```
models/model.pt
```

relative to your project root (i.e. a `models/` folder sitting next to
`app/`, `notebooks/`, etc. — not inside `app/`).

```bash
streamlit run app/streamlit_app.py
```

Upload a "before" (T1) and "after" (T2) tile to see:
- Predicted class + confidence + top-3 predictions, for each tile
- Cosine similarity between their embeddings
- A **Detection Mode** toggle (sidebar) to switch the similarity threshold
  between High Recall / Balanced / High Precision
- A change/no-change flag based on the selected threshold
- A side-by-side pixel-difference heatmap

No internet connection is needed to run the dashboard once the checkpoint
is in place.

## 📖 Project Approximations

### Spatial Block Split
Standard EuroSAT has no geographic coordinates, so the "spatial block"
split groups each class's images into contiguous chunks (assumed to
approximate nearby tiles) instead of a random shuffle.

### Temporal Change Simulation
EuroSAT has no real before/after imagery. Notebook 4 simulates T1/T2 pairs
from contiguous per-class chunks, defining "changed" as a class change
between the two tiles in a pair.

Both approximations are explained in detail inside their respective
notebooks.
